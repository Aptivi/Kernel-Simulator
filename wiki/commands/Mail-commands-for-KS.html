    Brief overview of mail shell

    The mail shell, invoked by mail, is one of the shells available in the kernel that provides you basic mail functions, such as reading mail, reading encrypted mail, sending mail, listing mail, and so on. It supports folder management.

    For more information about every command, click the command.

    All mail commands

    Command Description
    cd Changes directory to another mail folder.
    exit Exits the mail shell and returns to shell.
    list Lists messages. You can use page number to navigate your messages.
    lsdirs Lists directories.
    mkdir Makes a mail directory
    mv Moves a message to another directory.
    mvall Moves all messages from a specified sender to another directory.
    read Reads a message.
    readenc Reads an encrypted message. You should have the sender's GPG public key.
    ren Renames a directory.
    rm Removes a message.
    rmall Removes all messages from a specified sender.
    rmdir Removes a mail directory
    send Sends mail.
    sendenc Sends encrypted mail. You should have your own private key, and the recipient should have your public key.
