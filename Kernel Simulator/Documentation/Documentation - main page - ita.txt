| --------------- + -> Kernel Simulator - Simula il futuro kernel pianificato da noi <- + -------------- - |

INFORMAZIONI: Stai creando una versione Linux di Kernel Simulator in MonoDevelop? In tal caso, installare MonoDevelop e libmono-microsoft-visualbasic10.0-cil. Mentre suggeriamo MonoDevelop, raccomandiamo JetBrains Rider per Linux a 64 bit.

INFORMAZIONI: Questo simulatore del kernel non è una versione finale pianificata di Kernel, poiché era minima.

INFORMAZIONI: Due documentazioni, domande frequenti e risoluzione dei problemi, vengono eliminate perché non vengono aggiornate regolarmente.

Kernel Simulator ti consente di simulare un accesso molto precoce per il nostro futuro Kernel che continuerà a essere sviluppato, anche se abbiamo realizzato la versione PC reale di Kernel.
