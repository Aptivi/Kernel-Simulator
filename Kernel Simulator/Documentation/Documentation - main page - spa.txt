| --------------- + -> Kernel Simulator - Simula el kernel futuro que planeamos nosotros <- + -------------- - |

INFO: ¿Está creando una versión para Linux de Kernel Simulator en MonoDevelop? Si es así, instale MonoDevelop y libmono-microsoft-visualbasic10.0-cil. Mientras sugerimos MonoDevelop, recomendamos JetBrains Rider para Linux de 64 bits.

INFORMACIÓN: Este simulador de kernel no es una versión final planificada de Kernel, ya que era mínima.

INFO: Se eliminan dos documentaciones, preguntas frecuentes y solución de problemas porque no se actualizan periódicamente.

Kernel Simulator le permite simular un acceso muy temprano para nuestro futuro Kernel que continuará desarrollándose, incluso si creamos la versión real para PC de Kernel.
