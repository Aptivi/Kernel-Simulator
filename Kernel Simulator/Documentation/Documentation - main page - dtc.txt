| --------------- + -> Kernel Simulator - Simuleert de toekomstige kernel die door ons is gepland <- + -------------- - |

INFO: Bouw je een Linux-versie van Kernel Simulator in MonoDevelop? Installeer in dat geval MonoDevelop en libmono-microsoft-visualbasic10.0-cil. Hoewel we MonoDevelop voorstellen, raden we JetBrains Rider voor Linux 64-bit aan.

INFO: Deze kernelsimulator is geen definitieve geplande versie van Kernel, omdat deze minimaal was.

INFO: Twee documentatie, faq en probleemoplossing, worden verwijderd omdat ze niet regelmatig worden bijgewerkt.

Met Kernel Simulator kunt u een zeer vroege toegang simuleren voor onze toekomstige Kernel die nog zal worden ontwikkeld, zelfs als we de echte pc-versie van Kernel hebben gemaakt.
