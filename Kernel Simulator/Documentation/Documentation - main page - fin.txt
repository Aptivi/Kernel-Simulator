| --------------- + -> Kernel-simulaattori - simuloi tulevaa ydinä, jonka suunnittelemme <- + -------------- - |

INFO: Rakennatko Linux-versiota Kernel Simulatorista MonoDevelopissa? Jos näin on, asenna MonoDevelop ja libmono-microsoft-visualbasic10.0-cil. Vaikka suosittelemme MonoDevelopia, suosittelemme JetBrains Rideriä Linux 64-bittiseksi.

INFO: Tämä ytimen simulaattori ei ole lopullinen suunniteltu ytimen versio, koska se oli minimaalinen.

INFO: Kaksi dokumentaatiota, vika ja vianmääritys, poistetaan, koska niitä ei päivitetä säännöllisesti.

Ytimen simulaattorin avulla voit simuloida hyvin varhaisen pääsyn tulevaan ytimellemme, jota kehitetään edelleen, vaikka teimme Kernelin todellisen PC-version.
