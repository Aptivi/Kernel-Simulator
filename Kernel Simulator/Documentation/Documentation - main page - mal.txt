| --------------- + -> Kernel Simulator - Simulasikan kernel masa depan yang dirancang oleh kami <- + -------------- - |

INFO: Adakah anda membina versi Simulator Kernel Linux dalam MonoDevelop? Jika ya, pasangkan MonoDevelop dan libmono-microsoft-visualbasic10.0-cil. Walaupun kami mencadangkan MonoDevelop, kami mengesyorkan JetBrains Rider untuk Linux 64-bit.

INFO: Simulator kernel ini bukan versi akhir Kernel yang dirancang, kerana ia adalah minimum.

INFO: Dua dokumentasi, faq dan penyelesaian masalah, dipadam kerana ia tidak dikemas kini secara berkala.

Simulator Kernel membolehkan anda mensimulasikan akses awal untuk Kernel masa depan kami yang akan terus dibangunkan, walaupun kami membuat versi PC sebenar Kernel.
