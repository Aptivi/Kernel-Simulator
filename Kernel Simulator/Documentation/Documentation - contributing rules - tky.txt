- Çekirdek Simülatörüne katkıda bulunmak istiyorsanız, aşağıdaki şartlara uyulmalıdır:

1. Çekirdek çekirdeğini değiştirmek / yeniden kodlamak istiyorsanız, değişiklikleri doğru ve dikkatlice yazdığınızdan ve çekirdek çekirdeği özelliklerine zarar vermediğinizden emin olun.
2. İstediğiniz bazı özellikleri her yere eklemek / değiştirmek / kaldırmak / yeniden kodlamak istiyorsanız, değişiklikleri dikkatlice yazın.
3. Çekirdek Simülatörüne katkıda bulunmak isteyen diğer geliştiriciler gibi, değişiklikler de test edilmelidir. Sonuçları istemiyoruz, sadece test ediyoruz ve test edildiğini söylüyoruz. Sonuçları vermek istiyorsanız, çekme isteğinizi yine de kabul ediyoruz.
4. Eğer test etmediyseniz ve bize çekme isteği gönderdiyseniz, kodu test edeceğiz ve şunu bulursak:

   - kodunuz çalışıyor, çekme isteğinizi kabul edeceğiz
   - kodunuz çalışmıyor, çekme isteğinizi kapatacağız

5. Test etmediyseniz ve bize göndermeye hazırsanız, önce kodu test edin.
6. Dokümanları veya kodlama dışındaki diğer parçaları değiştirmek veya / yeni dokümantasyonun parçalarını eklemek istiyorsanız, çekinmeyin. Bize bildirebilirsiniz.

- Diğer işletim sistemlerine veya Assembly (ASM, NASM, ...) veya C / C ++ dilden farklı programlama dillerine taşıma:

Çekirdek Simülatörü'nü Linux, macOS vb. Diğer işletim sistemlerine aktarmanız gerekebilir. Ve ayrıca, C / C ++ veya Assembly dilden başka bir programlama diline.

ASM / C / C ++ 'a taşınması bize aittir. Resmi Çekirdek ASM / C / C ++ önyüklenebilir sürümümüzden önceki herhangi bir bağlantı noktası bizim tarafımızdan desteklenmeyecek ve bunlarla ilgili sorunlar derhal reddedilecektir.

- Çekirdek Simülatörü'nü Visual Basic'te Linux'a (test edilmiş) veya macOS'a (denenmemiş) taşımak istiyorsanız:

1. Mono, MonoDevelop ve libmono-microsoft-visualbasic10.0-cil paketlerini kurun. Onların resmi web sitesinde nasıl olduğunu bulabilirsiniz.
2. KS kaynak kodunu indirin
3. MonoDevelop'u kullanarak çözümü açın ve programı çalıştırın.
4. KS depomuzdaki Sorunlar bölümündeki sorunları bildirin.

- Çekirdek Simülatörü'nü Sun OS, BSD'ler, Solaris vs.'ye (denenmemiş) bağlamak istiyorsanız:

Bu bölüm yapım aşamasındadır. Talimatlarınız varsa, lütfen katkıda bulunun.

Not: Çekirdek Simülatörü'nü Assembly veya C / C ++ ile eşit olmayan başka bir programlama diline taşımak istiyorsanız, çekinmeyin.
Not: Çekirdek Simülatörü içindeki Çekirdeği, Montaj ve C / C ++ kullanarak Gerçek Bilgisayar / Sanal Kutu'ya taşımak bize kalmış.
Not: Simülatördeki Çekirdeği Android / iOS'a Taşımak Tamam. Uygun değişiklikleri yapabilirsiniz.
