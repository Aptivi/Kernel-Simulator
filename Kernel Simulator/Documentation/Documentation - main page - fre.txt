| --------------- + -> Kernel Simulator - Simule le futur noyau que nous avons planifié <- + -------------- - |

INFO: Vous construisez une version Linux de Kernel Simulator dans MonoDevelop? Si tel est le cas, installez MonoDevelop et libmono-microsoft-visualbasic10.0-cil. Bien que nous suggérons MonoDevelop, nous recommandons JetBrains Rider pour Linux 64 bits.

INFO: Ce simulateur de noyau n'est pas une version finale prévue du noyau, car il était minimal.

INFO: Deux documentations, la FAQ et le dépannage, sont supprimés car ils ne sont pas mis à jour régulièrement.

Kernel Simulator vous permet de simuler un accès très précoce pour notre futur noyau, qui continuera à être développé, même si nous avons créé la version PC réelle du noyau.
