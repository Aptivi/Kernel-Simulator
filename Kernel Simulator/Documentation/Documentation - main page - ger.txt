| --------------- + -> Kernel Simulator - Simuliert den von uns geplanten zukünftigen Kernel <- + -------------- - |

INFO: Erstellen Sie eine Linux-Version von Kernel Simulator in MonoDevelop? Wenn ja, installieren Sie MonoDevelop und libmono-microsoft-visualbasic10.0-cil. Während wir MonoDevelop empfehlen, empfehlen wir JetBrains Rider für Linux 64-Bit.

INFO: Dieser Kernel-Simulator ist keine endgültige geplante Version von Kernel, da er minimal war.

INFO: Zwei Dokumentationen (FAQ und Fehlerbehebung) werden gelöscht, da sie nicht regelmäßig aktualisiert werden.

Mit dem Kernel-Simulator können Sie einen sehr frühen Zugriff für unseren zukünftigen Kernel simulieren, der weiterentwickelt wird, auch wenn wir die echte PC-Version von Kernel erstellt haben.
