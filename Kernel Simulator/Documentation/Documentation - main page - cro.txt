INFO: Gradite li Linux verziju Kernel Simulatora u MonoDevelopu? Ako je to slučaj, instalirajte MonoDevelop i libmono-microsoft-visualbasic10.0-cil. Iako predlažemo MonoDevelop, za 64-bitni Linux preporučujemo JetBrains Rider.

INFO: Ovaj simulator kernela nije konačna planirana verzija Kernela jer je bio minimalan.

INFO: Izbrišu se dvije dokumentacije, faq i rješavanje problema jer se ne ažuriraju redovito.

Kernel Simulator omogućuje vam simuliranje vrlo ranog pristupa za naš budući Kernel koji će se i dalje razvijati, čak i ako smo napravili pravu PC verziju Kernela.
