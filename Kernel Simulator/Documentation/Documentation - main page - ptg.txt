| --------------- + -> Kernel Simulator - Simula o futuro kernel que é planejado por nós <- + -------------- - |

INFO: Você está construindo uma versão do Linux do Kernel Simulator no MonoDevelop? Em caso afirmativo, instale MonoDevelop e libmono-microsoft-visualbasic10.0-cil. Apesar de sugerirmos o MonoDevelop, recomendamos o JetBrains Rider for Linux 64-bit.

INFO: Este simulador de kernel não é uma versão final planejada do Kernel, já que era mínima.

INFO: Duas documentações, faq e resolução de problemas, são excluídas porque não são atualizadas regularmente.

O Kernel Simulator permite simular um acesso muito antecipado para o nosso futuro Kernel que continuará a ser desenvolvido, mesmo se fizermos a versão real do Kernel para PC.
