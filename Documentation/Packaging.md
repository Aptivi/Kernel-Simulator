
# Packaging

The packing and distribution procedures are now easier <br>
by executing this script below on the command line.

<br>

## Steps

<br>

1.  Open a new terminal in the root directory of **KS**

    <br>

2.  Execute the following to pack:

    <br>

    ### Linux

    ```shell
    ./buildandpack.sh
    ```
    
    <br>
    
    ### Windows
    
    *with `cmd`*
    
    ```shell
    buildandpack
    ```

<br>
