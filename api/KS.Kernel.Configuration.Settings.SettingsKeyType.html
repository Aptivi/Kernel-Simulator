    Assembly: Nitrocid.dll
    Syntax
    public enum SettingsKeyType

    Fields

    Name Description
    SBoolean

    The value is of bool
    SChar

    The value is of char and only accepts one character.
    SColor

    The value is of Terminaux.Colors.Color and comes from the color wheel
    SDouble

    The value is of double
    SFiglet

    The value is a Figlet font name that allows Settings to use the Figlet font selector.
    SInt

    The value is of int
    SIntSlider

    The value is of int, but has a slider which has a minimum and maximum value. Useful for numbers which are limited.
    SList

    The value is of IEnumerable<T>, like List<T>
    SPreset

    The value is a shell preset defined using IPromptPreset in CurrentPresets
    SSelection

    The value is of the selection, which can either come from enums, or from IEnumerable<T>, like List<T>
    SString

    The value is of string
