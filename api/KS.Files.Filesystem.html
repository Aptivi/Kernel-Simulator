ThrowOnInvalidPath(String)

Mitigates Windows 10/11 NTFS corruption/Blue Screen of Death (BSOD) bug

Declaration
public static void ThrowOnInvalidPath(string Path)
Parameters
Type Name Description
System.String Path

Target path
Remarks
  • When we try to access the secret NTFS bitmap path, which contains $i30, from the partition root path, we'll trigger the "Your disk is corrupt."
  • When we try to access the kernelconnect secret device from the system partition root path, we'll trigger the BSOD.

    This sub will try to prevent access to these paths on unpatched systems and patched systems by throwing System.ArgumentException
